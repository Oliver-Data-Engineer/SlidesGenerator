# Cores — Itaú Empresas

> Valores verificados contra o Guia de Marca V3. Para conferir: `refs/guia-marca/20250627_IE_REDUZ_GuiaMarca_V3.pdf`

---

## Paleta Principal

### Laranja Itaú (Primária)
| Formato | Valor |
|---------|-------|
| HEX | `#EC7000` |
| RGB | `rgb(236, 112, 0)` |
| HSL | `hsl(28, 100%, 46%)` |
| CSS var | `var(--color-brand-primary)` |

**Uso:** CTAs, destaques, bordas de ênfase, ícones, gráficos principais, fundo de slide de título.

---

### Azul Escuro Itaú (Secundária)
| Formato | Valor |
|---------|-------|
| HEX | `#000D3C` |
| RGB | `rgb(31, 59, 107)` |
| HSL | `hsl(218, 55%, 27%)` |
| CSS var | `var(--color-brand-secondary)` |

**Uso:** Fundos escuros, textos em fundos claros, cabeçalhos, barras de progresso.

## Neutros

| Nome | HEX | CSS var | Uso |
|------|-----|---------|-----|
| Branco | `#FFFFFF` | `var(--color-neutral-white)` | Fundos, textos em fundo escuro |
| Cinza 100 | `#F5F5F5` | `var(--color-neutral-100)` | Fundos alternativos, cards |
| Cinza 300 | `#CCCCCC` | `var(--color-neutral-300)` | Bordas, divisores |
| Cinza 500 | `#999999` | `var(--color-neutral-500)` | Textos secundários, notas |
| Cinza 700 | `#555555` | `var(--color-neutral-700)` | Textos de corpo |
| Preto | `#1A1A1A` | `var(--color-neutral-black)` | Títulos, textos de alta ênfase |

## Cores de Apoio (Semânticas)

| Nome | HEX | Uso |
|------|-----|-----|
| Sucesso | `#2E7D32` | Indicadores positivos, metas atingidas |
| Atenção | `#F57C00` | Alertas, pontos de atenção |
| Erro | `#C62828` | Falhas, pontos críticos |
| Info | `#1565C0` | Informações complementares |

---

## Regras de Combinação para Slides

| Cenário | Fundo | Texto | Destaque |
|---------|-------|-------|----------|
| Slide título dark | `#000D3C` | `#FFFFFF` | `#EC7000` |
| Slide título light | `#EC7000` | `#FFFFFF` | `#FFFFFF` |
| Slide de conteúdo | `#FFFFFF` | `#1A1A1A` | `#EC7000` |
| Slide de seção | `#1A1A1A` | `#FFFFFF` | `#EC7000` |
| Slide de código | `#000D3C` | `#F5F5F5` | `#EC7000` |

---

## Contraste WCAG

| Par | Razão | Nível |
|-----|-------|-------|
| `#EC7000` em `#FFFFFF` | 3.1:1 | AA (textos grandes) |
| `#000D3C` em `#FFFFFF` | 9.7:1 | AAA |
| `#FFFFFF` em `#EC7000` | 3.1:1 | AA (textos grandes) |
| `#FFFFFF` em `#000D3C` | 9.7:1 | AAA |
| `#1A1A1A` em `#FFFFFF` | 16.1:1 | AAA |

> Para textos de corpo (abaixo de 18px), prefira `#000D3C` sobre `#EC7000` como cor de texto.
